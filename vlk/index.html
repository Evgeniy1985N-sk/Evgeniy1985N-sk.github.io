Как удалить программу VLC media player?

После установки программного обеспечения VLC media player (далее именуемого Программа) Вы можете в любое время удалить ее с компьютера, используя стандартные функции Windows:

Для Windows XP, Vista, 7, 8, 8.1, 10

  1. Нажмите "Пуск" - "Панель управления";
  2. Выберите "Установка и удаление программ" или "Программы и компоненты";
  3. Выберите Программу VLC media player из списка с соответствующим логотипом и названием;
  4. Нажмите на кнопку "Удалить";
  5. Дождитесь появления диалогового окна программы деинсталятора, которая поможет удалить Программу;
  6. Дождитесь завершения всех шагов деинсталятора.

Возможно, для полного и правильного завершения процедуры удаления Программы потребуется перезагрузить компьютер.

Удаление дополнительного программного обеспечения

Если вы согласились с установкой дополнительного программного обеспечения и/или их компонентов и через какое-то время решите удалить эти программы с компьютера, можно будет сделать это стандартными средствами удаления и установки программ операционной системы Windows:

  1. Нажмите "Пуск", далее "Настройка", далее "Панель управления";
  2. Выберите "Установка и удаление программ" или "Программы и компоненты";

  4. Нажмите на кнопку "Удалить";
  5. Если операционная система Windows потребует перезагрузку - рекомендуем выполнить ее для полного удаления дополнительного программного обеспечения.
